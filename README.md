# Paste.ly
IntelliJ Plugin

Tabs (as in `\t`) are ignored as IDEA will put tabs in for you. Use `⇥` if you really want a tab.

# Supported tags

## Useful navigation and control

| Symbol | Description |
|--------|-------------|
| `§` | Start a section (each section will be individually pasted)
|`←` | Left
|`↑` | Up
|`→` | Right
|`↓` | Down
|`⌫` | Backspace
|`␛` | Esc
|`␡` | Del
|`⏎` | Enter
|`⇧` | Shift
|`⌘` | Cmd
|`␑` | Ctrl
|`⌥` | Option
|`⎇` | Alt
|`❙` | Pause (250ms)
|`⇥` | Tab
